## 三、逻辑正确性检查

### 3.1 核心算法验证

#### 3.1.1 结石等级计算逻辑
```dart
// UserData.stoneLevel getter
int get stoneLevel {
  if (currentStreak >= 7) return 0;  // 连续7天健康
  if (consecutiveMissedDays == 0) return 0;  // 无漏打健康
  if (consecutiveMissedDays == 1) return 1;  // 漏1天
  if (consecutiveMissedDays <= 3) return 2;  // 漏2-3天
  if (consecutiveMissedDays <= 6) return 3;  // 漏4-6天
  return 4;  // 漏7天+
}
```
**评价**：逻辑清晰，符合需求
